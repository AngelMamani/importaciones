# Maquinarias P&M - Sistema de Administración

Sistema web de administración para **Maquinarias P&M S.A.C.**, diseñado para la gestión centralizada de inventario industrial, cotizaciones y clientes.

## 🚀 Tecnologías
- **React 19** + **TypeScript**
- **Vite** (Build Tool)
- **React Router Dom** (Navegación)
- **Vanilla CSS** (Premium Design System)
- **Architecture**: Screaming Architecture (Modular y Escalable)

## 🎨 Características de Diseño
- Interfaz moderna e industrial basada en el sistema Blue & Gold.
- Layout completamente responsive (Móvil, Tablet, Escritorio).
- Navegación dinámica con Sidebar corporativo.
- Componentes de alta fidelidad con micro-animaciones.

## 📁 Estructura del Proyecto
- `src/features`: Módulos del negocio (Dashboard, Inventory, etc.)
- `src/layouts`: Estructura base del panel administrativo.
- `src/assets`: Recursos visuales y logotipos.

## 🛠️ Instalación y Desarrollo
1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar dependencias.
3. Ejecutar `npm run dev` para iniciar el servidor de desarrollo.
